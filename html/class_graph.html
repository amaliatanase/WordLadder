Detailed Description

template<typename T>
class Graph< T >

A generic template class representing an undirected graph.

  • Template Parameters
    TThe data type of the nodes stored in the graph (e.g., std::string). Uses an adjacency list implemented with a std::map and std::set for efficient neighbor lookups.
