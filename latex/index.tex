      \hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{🎮 Game Modes}{🎮 Game Modes}}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Automatic Mode:\+}} Instantly finds the optimal, shortest path between a starting word and a target word using the Breadth-\/\+First Search (BFS) algorithm.
\item {\bfseries{Playing Mode:\+}} Challenges the user to solve the ladder step-\/by-\/step. It includes strict real-\/time move validation and an intelligent, color-\/coded hint system.
\item {\bfseries{Analytics Mode:\+}} Tracks player performance across multiple sessions. It reads and parses CSV files to display personalized statistics, such as total games played and the number of unique words used.
\end{DoxyItemize}\hypertarget{index_autotoc_md3}{}\doxysection{\texorpdfstring{🛠️ Technologies \& Data Structures}{🛠️ Technologies \& Data Structures}}\label{index_autotoc_md3}

\begin{DoxyItemize}
\item {\bfseries{Language:\+}} C++
\item {\bfseries{GUI Framework:\+}} Qt
\item {\bfseries{Core Algorithm:\+}} Breadth-\/\+First Search (BFS) for optimal pathfinding and graph traversal.
\item {\bfseries{Custom Structures:\+}} A generic \doxylink{class_graph}{Graph} template implemented from scratch using {\ttfamily std::\+map} and {\ttfamily std::\+set} for \$\+O(1)\$/\+\$O(\textbackslash{}log n)\$ neighbor lookups.
\item {\bfseries{Data Management:\+}} File I/\+O operations and CSV parsing for persistent gameplay history.
