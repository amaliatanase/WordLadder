\doxysubsection{Detailed Description}
The main graphical user interface for the Word Ladder Game. 


\begin{DoxyItemize}
\item This class inherits from QMain\+Window and manages the three main tabs of the application:\+ Automatic Mode (finding paths automatically), Playing Mode (interactive gameplay), and Analytics Mode (viewing player statistics).
