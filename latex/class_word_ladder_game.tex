\doxysubsection{Detailed Description}
Handles the core logic and backend operations for the Word Ladder game. 


\begin{DoxyItemize}
\item This class is responsible for loading the dictionary, building the graph of words, generating valid game states, and running the Breadth-\/\+First Search (BFS) algorithm to find optimal paths. 
\end{DoxyItemize}
